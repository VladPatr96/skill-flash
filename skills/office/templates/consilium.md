# Консилиум: {{тема}}

## Протокол (ведёт оркестратор)

1. **Вопрос** — сформулируй нейтрально, без своего мнения и без подсказки предпочтения. Собери одинаковый контекст-пакет всем участникам (факты, ограничения, `graphify query`-выжимки).
2. **Независимые мнения** — 2–4 параллельно, участники НЕ видят ответы друг друга:
   - deep-reasoner (Opus),
   - Codex (`/codex:rescue --background`),
   - опционально ещё deep-reasoner с другой линзой (риски / стоимость / пользователь) или дизайн-агент для визуальных развилок.
3. **Анонимизация** — мнения становятся A, B, C (без имён моделей), чтобы не взвешивать по авторитету.
4. **Извлечение утверждений** — разбери мнения на атомарные утверждения; по каждому отметь, кто согласен/против.
5. **Консенсус** — совпавшие утверждения идут в решение. По расхождениям: либо взвешиваешь сам, либо второй раунд — каждому участнику показать чужие аргументы (без авторства) с вопросом «что из этого меняет твою позицию?». Максимум 2 раунда; не сошлось — решаешь сам и фиксируешь разногласие. Тай-брейкер: при прочих равных побеждает более простое и дешёвое решение — недостающее доделывается отдельным issue потом.
6. **Итог** → в секцию «Решённые развилки» спеки + протокол в `company/meetings/YYYY-MM-DD-consilium-{{тема}}.md`.
