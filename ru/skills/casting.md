# casting

Используй, когда для задачи нужен новый отдел, агент, tool или skill.

Найм начинается с конкретного use case. Сначала проверь registry и существующие skills, затем verified repos/marketplaces. Оффер должен включать department, charter, tool/agent, model class, first task, cost/value, risks и fallback.

Установки, paid services, uploads, external accounts и новые отделы требуют approval.
