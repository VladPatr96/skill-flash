# Corporate

Report progress toward outcomes, not activity volume.

## Procedure

1. Name source of truth for every KR. No source means `UNKNOWN`.
2. Mark each claim as `FACT`, `HYPOTHESIS`, or `UNKNOWN`.
3. Show OKR progress, roadmap gates, and possible Goodhart traps.
4. Compare hiring/tool/stage offers against actual cost and value.
5. Check pipeline health: stale issues, blockers, verifier failures, overloaded departments, stale docs/graphs.
6. Present at most five founder decisions with recommendation, reason, and cost.

## Output

Write or update `company/reports/YYYY-MM-review.md`, then update `company/state.md`, `company/okr.md`, roadmap, or issues when findings require action.

No metric without source of truth.
